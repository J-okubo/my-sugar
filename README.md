### My Sugar

My Sugar は血糖値管理をサポートする Web アプリです。

### 環境構築

# 必要なもの
-Docker
-Laravel sail
-Vite

### インストール
git clone https://github.com/your-repo/my-sugar.git

### ディレクトリ移動
cd my-sugar

### Dockerコンテナを起動
./vendor/bin/sail up -d  

### nodeをインストール
./vendor/bin/sail npm install  

### ブラウザを立ち上げ
./vendor/bin/sail npm run dev
